Welcome to ACM's Web Development Workshop.

We shall be going through a 2 day intensive web development course covering HTML, CSS and a framework called Bootstrap . We shall begin our learning with HTML 5. This is the latest functional increment in the HTML. The goal of HTML is to reduce the amount of scripting required in a webpage by introducing most of the functionalities in terms of tags.
Then we move on to CSS which is a language that works with HTML. It describes the style of the HTML webpage. Then we'll move on to BootStrap which is a framework for web development based on HTML and CSS classes.
